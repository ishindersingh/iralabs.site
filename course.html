What You'll Build

1. Smart Light

Level: Beginner
Learn the basics of inputs and outputs by building a light that turns on automatically in the dark.

2. Reverse Parking Sensor

Level: Beginner
Master ultrasonic sensors by creating a device that beeps faster as an object gets closer.

3. Line Follower Robot

Level: Intermediate
Build a robot that can autonomously follow a black line on a white surface using IR sensors.

4. Smart Home System

Level: Intermediate
Create an IoT-based system to control home appliances wirelessly using your smartphone.

5. Obstacle Avoiding Robot

Level: Advanced
Combine motors and sensors to build a fully autonomous robot that navigates without hitting obstacles.

6. WiFi Surveillance Bot

Level: Advanced
Build a remote-controlled robot with a live camera feed that you can drive from anywhere in the world.
